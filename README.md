# Bank Management System - Learning Project

This is a **learning project** to practice full‑stack Java web development with Spring Boot, JSP, JDBC, and PostgreSQL.

## What it does

- Users can **log in** to the system.
- View **bank account** details and **transaction history**.
- Perform **deposit** and **withdraw** operations.
- See a clean dashboard with a professional frontend design.

## Technologies used

- **Backend**: Java, Spring Boot, Spring MVC, Spring JDBC
- **Frontend**: JSP, CSS
- **Database**: PostgreSQL
- **Tools**: Git, GitHub (this repo)

## How to run locally

1. Install Java 17, Maven, and PostgreSQL.
2. Create a database named `bankdb` (or configure a different name in `application.properties`).
3. Run the project with:
   ```bash
   mvn spring-boot:run
   ```
4. Open the login page in your browser (usually `http://localhost:8080`).
5. Use the demo login (if any) shown in the project’s `data.sql` or `README`.

## Purpose

This project is for **learning** and **practice**, not for production use. It helps to:
- Understand how to connect Spring Boot with JDBC and PostgreSQL.
- Learn session‑based authentication and CRUD operations.
- Get a clean, simple project you can showcase in your portfolio or resume.
